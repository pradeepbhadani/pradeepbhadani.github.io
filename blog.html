QuickStart : Install Chef-Server 12

Install Chef Server 12 on RHEL 6.x

Prerequisite

Machine should resolve hostname

  1. Get Chef server 12 RPM for RHEL 
wget https://web-dl.packagecloud.io/chef/stable/packages/el/6/chef-server-core-12.1.2-1.el6.x86_64.rpm -o chef-server-core-12.1.2-1.el6.x86_64.rpm
  1. Install chef-server 12
sudo rpm -Uvh chef-server-core-12.1.2-1.el6.x86_64.rpm
  1. Configure chef-server 12
sudo chef-server-ctl reconfigure
  1. Create admin user
sudo chef-server-ctl user-create <user_name> <first_name> <last_name> <email> <password> --filename <FILE_NAME
sudo chef-server-ctl user-create admin admin_fname admin_lname admin@xyz.email adminpswd --filename /home/vagrant/admin.pem
  1. Create Organization
sudo chef-server-ctl org-create <short_name> <full_organization_name> --association_user <user_name> --filename <ORGANIZATION-validator.pem>
sudo chef-server-ctl org-create org "Test Org" --association_user admin --filename /home/vagrant/org-validator.pem
  1. Install Chef Manage
sudo chef-server-ctl install opscode-manage
sudo chef-server-ctl reconfigure
sudo opscode-manage-ctl reconfigure
  1. Install Chef push Job
sudo chef-server-ctl install opscode-push-jobs-server
sudo chef-server-ctl reconfigure
sudo opscode-push-jobs-server-ctl reconfigure
  1. Install Chef Reporting
sudo chef-server-ctl install opscode-reporting
sudo chef-server-ctl reconfigure
sudo opscode-reporting-ctl reconfigure
  1. Point your browser to http://<hostname/IP>/ to access Chef UI and use credentials generated in Step4
